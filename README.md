# Campus Life Super App

## What this project is
This is my Campus Life Super App — basically a one-stop web app for students to manage everyday campus stuff. It has **events**, **weather updates**, a **study timer**, and a **dining finder**, all in one place. My goal was to make something **useful and easy for students to use**.

---

## What I used to build it
- **HTML5** – to set up the pages  
- **CSS3** – for custom styles and responsive layout  
- **Bootstrap 5** – makes styling and responsive design easier  
- **JavaScript** – for all the interactive features (weather, timer, etc.)  
- **OpenWeatherMap API** – to get real-time weather info  
- **Git & GitHub** – for version control  
- **GitHub Pages** – to make the app live on the web  

---

## Features
1. **Homepage**  
   - Quick welcome message  
   - Buttons that take you to Events, Weather/Study Timer, and Dining  

2. **Events Page**  
   - Shows campus events  
   - Easy to go back to the homepage  

3. **Weather + Study Timer**  
   - Type in a city and see the current weather (temperature + description)  
   - Start a study timer for focused sessions, alert pops up when time’s up  

4. **Dining Finder**  
   - See campus dining options  
   - Makes planning meals easier
